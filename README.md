# 🐾 Pet Adoption Platform (Server)

## 📌 Project Name
Pet Adoption Platform API

## 🎯 Purpose
Backend API for Pet Adoption Platform built using Node.js, Express, and MongoDB. Handles authentication, pet management, and adoption request processing securely.

---

## 🚀 Live API URL
(https://assignment-9-server-1-cxv0.onrender.com/pets)

---

## ⚙️ Tech Stack

- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Cookie Parser
- CORS

---

## ✨ Features

- 🔐 JWT authentication with HTTPOnly cookies
- 👤 User registration and login system
- 🐶 CRUD operations for pets
- 📩 Adoption request system
- ✅ Approve / Reject requests
- 🔒 Protected routes using middleware
- 🗄️ MongoDB database integration
- ⚡ Secure environment variable setup
- 🚫 Prevent unauthorized access to routes

---

## 📦 NPM Packages Used

- express
- mongoose
- cors
- dotenv
- jsonwebtoken
- cookie-parser
- bcryptjs
